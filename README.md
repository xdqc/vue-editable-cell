# SYSDM


### To start REST API server:

```

	npm install
	npm run dev

	http://localhost:3000/api/vendors
```


### To test front-end:

	1. Go to ./www directory (put all frond-end stuff inside this folder)

```
	cd www
```


	2. Project setup
```
	npm install
```


	3. Compiles and hot-reloads in development mode
```
	npm run serve
```


	4. Open browser
```
	http://localhost:8081/
```
